# Subscription App

Простое Flutter приложение с функционалом онбординга, paywall экрана и подписки.

## Функционал

1. **Онбординг** - Экран приветствия с кнопкой "Продолжить"
2. **Paywall** - Экран выбора подписки (Месяц / Год со скидкой)
3. **Главный экран** - Контент для подписчиков
4. **Сохранение состояния** - Подписка сохраняется в SharedPreferences, при следующем запуске пользователь сразу попадает на главный экран

## Установка и запуск

### Вариант 1: Flutter версия (требует установки Flutter SDK)

1. Убедитесь, что у вас установлен Flutter SDK (см. `INSTALLATION.md`)
2. Установите зависимости:
   ```bash
   flutter pub get
   ```
3. Запустите приложение:
   ```bash
   flutter run
   ```

### Вариант 2: Веб-версия (без установки)

Откройте файл `web_version/index.html` в браузере. Все функции работают аналогично Flutter версии, используя localStorage для сохранения состояния.

## Структура проекта

```
lib/
  ├── main.dart                 # Точка входа, проверка подписки
  └── screens/
      ├── onboarding_screen.dart  # Экран онбординга
      ├── paywall_screen.dart     # Экран выбора подписки
      └── home_screen.dart        # Главный экран
```

## Особенности

- Эмуляция покупки подписки (без реального биллинга)
- Сохранение состояния через SharedPreferences
- Автоматическая навигация в зависимости от статуса подписки
- Простой и современный UI дизайн

## Тестирование

Для тестирования функционала:
1. При первом запуске откроется онбординг
2. После онбординга - paywall экран
3. Выберите план и нажмите "Продолжить" - откроется главный экран
4. Закройте и снова откройте приложение - главный экран откроется сразу
5. Нажмите кнопку выхода в главном экране для сброса подписки
